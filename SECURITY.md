# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in SeedVault:

1. **Do NOT open a public GitHub issue**
2. Email: **contact@seed-vault.io**
3. Include detailed steps to reproduce
4. Allow 48 hours for initial response

## Scope

- Encryption / decryption logic
- Key derivation (PBKDF2)
- QR code generation and scanning
- Steganography implementation
- Any data leakage (network, storage, clipboard)

## Out of Scope

- Brute-force attacks on weak passwords (user responsibility)
- Attacks requiring physical access to an unlocked device
- Social engineering
